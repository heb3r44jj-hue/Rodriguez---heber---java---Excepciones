# Sistema de Registro de Usuarios

Aplicación de consola desarrollada en Java que permite registrar usuarios validando la información ingresada y manejando excepciones personalizadas.

## Descripción

El sistema solicita el nombre, edad, correo electrónico y salario de un usuario, valida cada dato y muestra un mensaje indicando si el registro fue exitoso o si ocurrió algún error.

## Archivos del proyecto

- `Main.java` — Clase principal con el menú y la lógica del programa
- `NombreInvalidoExcepciones.java` — Excepción personalizada para el nombre
- `EdadInvalidaExcepciones.java` — Excepción personalizada para la edad
- `CorreoInvalidoExcepciones.java` — Excepción personalizada para el correo
- `SalarioInvalidoExcepciones.java` — Excepción personalizada para el salario

## Cómo ejecutar el programa

1. Abrir la terminal y ubicarse en la carpeta donde están los archivos
2. Compilar todos los archivos con el siguiente comando: javac*.java
3. Ejecutar el programa con el siguiente comando: java Main
